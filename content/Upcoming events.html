Will be presented by Mitch Rudge.

Bio Mitch is in the final year of a PhD with the Sustainable Minerals Institute, where he surveys savanna trees with drones to inform mine site restoration. To process and analyse drone data - LiDAR, photogrammetry, multispec - Mitch has been forced to pick up skills in geospatial analysis using R and python.

Workshop description For this session, Mitch will walk through the basics of raster analysis using R. In particular, he will focus on the relatively new Terra package, which is set to replace the extremely popular Raster package. A basic understanding of R will help, but this workshop will be aimed at beginners with no prior knowledge.
